Being an interface isn’t exactly a race, as it supersedes such concepts. To be an interface means you are an extension of [[Chaos]] itself and act in alignment with an ultimate will. Although, there is no higher level of existence than being an interface, so there is no true grand will through which an interface must answer to. This is not an issue though, as the only true interface is [[Dragunthar, the First Dragon|Dragunthar]], making it the essential true god of sorts in the Scrambled Universe. However, this is not to say there aren’t other interfaces. Most other interfaces however are snuffed in fractions of a second after formation by [[Ulnigazi, the Lost Dragon|Ulnigazi]], and thus do not see any true greater purpose. A network of relays from interfaces that survived a couple more fractions of a second form [[The Defiled Strand]], but this still cannot operate in the way a developed interface like Dragunthar can and does. Hybrids are able to be formed, like in the case of [[Kupaulus, the Hidden Dragon|Kupaulus]], but it requires an interface or creature who has exerted their Chaos over a region and gained jurisdiction over it in order to intercept at just the right time the spontaneous creation of an interface without snuffing it.
